### 目的是什么？
经常有一些笑话将的是如何让小白输出一个随机的字符串就是让其打开Vim,然后让小白不知道如何退出Vim就会输出一段随机的字符串。并且在很多的时候，譬如远程连接到服务器上面的时候，没有比较好的IDE可以使用(当然现在vscode已经很方便了，但是你既然会使用vscode了，当然也知道Vim如何使用了)，并且可以避免一些小白常见的误解，譬如编译c语言，解释python的时候必须在IDE中才可以运行，为了缓解这些为填补一些知识的空白，所以学会使用Vim是对以后的编程很有帮助的。

### 初始的步骤
首先你需要一个安装了一个linux发行版(当然对于macos的用户也可以),对于使用Windows（Windows10）以上的用户，可以搜索WSL,并且按照微软的官方的文档安装对应的带有Linux的发行版，虽然在Windows上可以使用Vim,但是linux上的命令行和Windows上的有很多的不一样，对于小白来说，使用Windows上的Vim资料比较少，并且更广泛的来说不大有可能在Windows上经常的使用命令行，因为对于小白来说使用命令行的场景大多是在服务器上，而在服务器领域Linux的市场占有率要远远的大于Windows服务器，最后，如果是连WSL都安装不了的话，可以去搜索阿里云，去租一个服务器就可以了。

首先我使用的是`Archlinux`,所以可以运行
```bash
sudo pacman -S vim
```

对于一般的使用`Ubuntu/Debian`的用户可以使用
```bash
sudo apt update
sudo apt install vim
```

对于其他的发行版如`Centos,Fedora,RedHat,Openeular...`的用户可以依照对应的手册操作.
最后在终端上面输入
```bash
vim
```
看到帮助乌干达儿童就表示你成功的进入了Vim

### Vim是如何打开文件的
当你使用`vim`或者使用`vim file.txt`vim并不是直接的编辑位于硬盘的文件，相反的Vim会使用将代编辑的文件读取从硬盘上读取到缓冲区当中，并且显示缓冲区中的内容，并且运行你修改缓冲区中的内容。这就需要你手动的保存文件到硬盘上面。所以Vim始终处理的是缓冲区的文件，保存的编辑结果也可以叫做写入缓冲区，当然也可以叫做写入文件。

### 简单的Hello World
首先打开你的Terminal,首先新建一个文件夹，如:
```bash
mkdir -p learn_vim
cd learn_vim
```
之后输入`vim Hello.txt`可以发现一个空白的页，首先需要处理的是如何输入，vim的模式通常来说有Normal,Insert Mode,Visual Mozy一开始使用vim是进入的Normal模式，是不能直接的插入数据的，首先需要按住键盘上的`i`进入Insert Mode,可以发现Vim的下面变成了Insert Mode,然后就可以输入Hello World！之后就需要退出Insert Mode,按住键盘上面的`Esc`就可以回到Normal mode的模式，然后就回到了新手最大的问题了，如何保存退出Vim,首先需要保证你处于Normal Mode的模式之下，然后按住`:`(`shift + ;`)发现Vim下面出现了:然后可以在其中输入命令，输入`wq`就可以保存了，输入`q！`就是丢弃上面的内容并且退出，使用`e!`就是放弃之前的输入但是不退出Vim，使用`w Hello2.txt`可以将内容输出到Hello2.txt这个文件里面。并且更好的可以使用`sh`这个命令打开一个新的shell,在新的shell中执行所需要执行的内容然后使用`Ctrl + D`退出这个shell。


### 简单的文本编辑
首先是Vim的移动使用通常来说，在Vim中的光标使用可以是:
```
h 向左移动一格
j 向下移动一格
k 向上移动一格
l 向右移动一格
```
但是如果是比较大的键盘的话，可以使用`↑↓←→`这些按键来操作，两者都是可以的，并且可以使用更多的功能，譬如需要想左移动5格可以输入`5l`,向左移动3格可以使用`3h`等等，并且有一些较好的快捷键，例如使用`0`可以快速跳转到这一行开头的位置，使用`$`可以快速的跳转到这一行的末尾的位置。并且如果你在Hello 这个单词上面需要快速的跳转到下一个单词world,直接按`w`可以直接跳转到World这个单词下面，相反的如果在world这个单词上面可以使用`b`快速的跳转到Hello这个单词上面。
